# EduSmart

EduSmart is a smart education web application designed to make learning more **engaging, flexible, and effective**.  
It provides a gamified learning experience with lessons, quizzes, progress tracking, and analytics.

## Features
- 🎓 **Interactive Lessons** – Multimedia-based content for easy learning.  
- 🧩 **Gamified Quizzes** – Multiple choice, fill-in-the-blank, and drag-and-drop questions.  
- 📊 **Analytics Dashboard** – Track daily progress and performance.  
- ⭐ **Progress Overview** – Resume where you left off with a "Continue Learning" section.  
- 🧑‍🎨 **User Profile** – XP tracking and personal learning journey.  

## Tech Stack
- **Frontend:** React, Tailwind CSS  
- **Backend (Planned):** Node.js, Express.js  
- **Database (Planned):** MongoDB
